1 General  
3DViewer v1.0 is a GUI application for viewing 3D wireframe models. The  
models themselves must be loaded from .obj les and be viewable on the screen  
with the ability to rotate, scale and translate.  
2 Installation  
You can build the programm right in the current directory from terminal by  
running make install , then run it as make run. Default installation: make  
install. To uninstall use make uninstall.  
3 Usage  
Main window of the application contains :  
A button to select the model le and a eld to output its name.  
A visualisation area for the wireframe model.  
Button/buttons and input elds for translating the model.  
Button/buttons and input elds for rotating the model.  
Button/buttons and input elds for scaling the model.  
Information about the uploaded model - le name, number of vertices and  
edges.
