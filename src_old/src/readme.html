1 General  
3DViewer v1.0 is a GUI application for viewing 3D wireframe models. The  
models themselves must be loaded from .obj Ñles and be viewable on the screen  
with the ability to rotate, scale and translate.  
2 Installation  
You can build the programm right in the current directory from terminal by  
running make install , then run it as make run. Default installation: make  
install. To uninstall use make uninstall.  
3 Usage  
Main window of the application contains :  
Ò A button to select the model Ñle and a Ñeld to output its name.  
Ò A visualisation area for the wireframe model.  
Ò Button/buttons and input Ñelds for translating the model.  
Ò Button/buttons and input Ñelds for rotating the model.  
Ò Button/buttons and input Ñelds for scaling the model.  
Ò Information about the uploaded model - Ñle name, number of vertices and  
edges.
